Häufige Fragen

Welche Dateien kann MSG Viewer öffnen?

MSG Viewer öffnet .msg-Dateien, wie sie von Microsoft Outlook beim Exportieren von E-Mails erstellt werden.

Wie öffne ich eine .msg Datei?

  • Doppelklick auf die Datei im Finder
  • Rechtsklick → „Öffnen mit" → MSG Viewer
  • Drag & Drop in das App-Fenster
  • In der App: Menü „Datei" → „Öffnen" (⌘O)

Kann ich mehrere E-Mails gleichzeitig laden?

Ja. Wählen Sie mehrere Dateien aus oder öffnen Sie einen ganzen Ordner mit ⇧⌘O.

Werden meine E-Mails irgendwohin gesendet?

Nein. MSG Viewer arbeitet vollständig offline. Siehe Datenschutzerklärung.

Brauche ich Microsoft Outlook?

Nein. MSG Viewer ist unabhängig von Microsoft-Produkten und benötigt kein Outlook.

Frequently Asked Questions

What files can MSG Viewer open?

MSG Viewer opens .msg files, as created by Microsoft Outlook when exporting emails.

How do I open a .msg file?

  • Double-click the file in Finder
  • Right-click → "Open With" → MSG Viewer
  • Drag & drop into the app window
  • In the app: Menu "File" → "Open" (⌘O)

Can I load multiple emails at once?

Yes. Select multiple files or open an entire folder with ⇧⌘O.

Are my emails sent anywhere?

No. MSG Viewer works entirely offline. See Privacy Policy.

Do I need Microsoft Outlook?

No. MSG Viewer is independent of Microsoft products and does not require Outlook.
